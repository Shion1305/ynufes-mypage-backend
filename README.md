## 学祭マイページシステムについて
学祭マイページシステムは、学祭関係者が各種申請を行い、円滑に申請を管理するためのシステムです。

## 技術スタック
言語: Go

インフラ: Firebase

CI/CD: GitHub Actions, Firebase Emulator
